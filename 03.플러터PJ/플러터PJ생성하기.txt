[ 플러터 PJ 생성하기 ]
-> 플러터 명령어를 통해 개발표준 패키지를 설치하여 개발을 더 쉽고 용이하게 할 수 있도록 구글이 지원함

-> 참고: 표준 개발 프로젝트 패키지를 보일러-플레이트(boilerplate 표준문안)라고 한다.
         즉, 프로젝트 표준코드를 의미한다.

[ 설치 순서 ]
1. 설치할 위치로 이동한다.(cd 명령어)

2. cmd 명령어창에서 프로젝트 생성 플러터 명령어를 실행한다.
->>> flutter create 프로젝트 이름
((프로젝트 명명법))
1) 숫자로 시작불가, -등 특수문자 사용 불가
2) 일반적인 _(언더바)를 사용한 단어연결로 명명함
예) app_01_store
->> flutter create 이름

3. 설치후 프로젝트 파일로 이동한다(cd명령어)

4. 프로젝트 폴더 위치에서 모듈을 실행시킨다.
->> flutter run
-> option중 2번 크롬브라우저 선택 -> 모듈 실행하여 일정 시간 후 브라우저로 로딩 됨!!!

5. 플러터 모듈 종료
-> cmd창에서 ctrl+c 하면 창닫기 가능
-> cmd창에서 쓰레기통 아이콘
